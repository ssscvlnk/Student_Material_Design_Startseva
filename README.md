# Лабораторная работа №10-11. Создание приложения «Список студентов» с использованием Material Design
**Student Material Design** — это учебное Android-приложение, разработанное в рамках лабораторных работ №10-11 для изучения принципов Material Design. Приложение отображает список студентов с возможностью раскрытия подробной информации о каждом из них.
## Цель работы
Научиться применять принципы Material Design в Androidприложениях с использованием Jetpack Compose. В результате лабораторной работы
будет создано приложение «Список студентов», которое демонстрирует использование
цветовых схем, типографики, форм и анимаций Material Design.
## Функциональность

Приложение представляет собой экран со списком студентов и обладает следующим функционалом:
-   Отображение списка студентов в виде карточек.
-   Каждая карточка содержит:
    -   Аватарку студента (круглой формы).
    -   Имя и возраст студента.
    -   Кнопку для раскрытия/сворачивания дополнительной информации.
-   Плавная анимация раскрытия и сворачивания карточки с описанием студента.
-   Адаптация интерфейса под светлую и тёмную тему системы.
-   Верхняя панель приложения (TopAppBar) с логотипом и названием.
## Технологии

-   **Язык программирования:** Kotlin
-   **UI-фреймворк:** Jetpack Compose
-   **Дизайн-система:** Material Design 3
-   **Используемые библиотеки и компоненты:**
    -   `androidx.compose.material3:material3` — реализация Material Design 3 компонентов (Card, TopAppBar, Scaffold, иконки, цветовые схемы)
    -   `androidx.compose.material:material-icons-extended` — расширенный набор иконок Material (ExpandMore, ExpandLess)
    -   `androidx.compose.ui` — базовые компоненты UI (Image, Text, Row, Column)
    -   `androidx.compose.foundation` — фундаментальные компоненты (LazyColumn для списков, layout-элементы)
    -   `androidx.compose.runtime` — управление состоянием (remember, mutableStateOf)
    -   `androidx.compose.animation` — анимации (animateContentSize, Spring)
-   **Шрифты:** Google Fonts (Montserrat, Abril Fatface)
